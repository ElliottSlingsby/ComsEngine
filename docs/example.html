Example Scripts

Load.lua

Example scripts need to be registered before use.

Inside Load.lua

Script.register("Grid", "component/Grid.lua")
Script.register("Camera", "component/Camera.lua")
Script.register("Clock", "component/Clock.lua")
Script.register("Cube", "component/Cube.lua")

Empty.lua

Defines an empty script component template.

Grid.lua

Provides a large wireframe grid at the origin point.

local grid = Entity.create()
grid:add(Script)

grid:get(Script):create("Grid")

Camera.lua

Press the ESC key to lock the mouse and gain first-person style controls over the camera. Use WASD to move.

local camera = Entity.create()
camera:add(Script)

camera:get(Script):create("Camera")

Clock.lua

Measures the FPS of the current simulation and prints out information every second.

local clock = Entity.create()
clock:add(Script)

clock:get(Script):create("Clock")

Cube.lua

Creates a randomly positioned entity that destroys itself after a random period of time, and creates a new entity upon destruction.
Used in large quantities to stress test the engine.

math.randomseed(os.time())

local i = 0
while i < 1024 do
    local cube = Entity.create()

    cube:add(Script)

    cube:get(Script):create("Cube")

    i = i + 1
end
