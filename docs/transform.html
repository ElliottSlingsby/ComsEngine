Transform (component object)

entity:add(Transform c, Vec3 position = (0, 0, 0), Quat rotation = (0, 0, 0, 1), Vec3 scale = (1, 1, 1))

Provides 3D positional data to an entity as to provide information to components that require rendering.

The transform component can be added with the constructor arguments for the initial position, rotation, and scale, all of which are set by default.

local entity = Entity.create()

entity:add(Transform, Vec3(0, 0, 0), Quat(Vec3(180, 0, 0)), Vec3(1, 1, 1))

Vec3 transform:position() | transform:position(Vec3 position)

Returns or sets the position using a Vec3.
The default position is (0, 0, 0)

local entity = Entity.create()
entity:add(Transform)
local transform = entity:get(Transform)

position = transform:position()

transform:position(Vec3(0, 0, 0))

Quat transform:rotation() | transform:rotation(Quat rotation)

Returns or sets the rotation using a Quat.
The default rotation is (0, 0, 0, 1)

local entity = Entity.create()
entity:add(Transform)
local transform = entity:get(Transform)

local rotation = transform:rotation()

transform:rotation(Quat(Vec3(0, 0, 0)))

Vec3 transform:scale() | transform:scale(Vec3 scale)

Returns or sets the scale using a Vec3.
The default scale is (1, 1, 1)

local entity = Entity.create()
entity:add(Transform)
local transform = entity:get(Transform)

local scale = transform:scale()

transform:scale(Vec3(2, 2, 2))

transform:translate(Vec3 translation) | transform:localTranslate(Vec3 translation)

transform:translate translates the position relative to the global rotation.

transform:localTranslate translates the position relative to the transform's rotation.

local entity = Entity.create()
entity:add(Transform)
local transform = entity:get(Transform)

transform:translate(Vec3(0, 0, 1) * Engine.dt())

transform:localTranslate(Vec3(0, 0, 1) * Engine.dt())

transform:rotate(Quat rotation) | transform:localRotate(Quat rotation)

transform:rotate performs a rotation on the global axes.

transform:localRotate performs a rotation relative to the axes of the transform's rotation.

local entity = Entity.create()
entity:add(Transform)
local transform = entity:get(Transform)

transform:rotate(Quat(Vec3(0, 90, 0)))

transform:localRotate(Quat(Vec3(0, 90, 0)))

transform:lookAt(Vec3 target, Vec3 up)

Changes a transform's position and rotation to look at a target.

local entity = Entity.create()
entity:add(Transform)
local transform = entity:get(Transform)

transform:lookAt(Vec3(0, 0, 0), Vec3(0, 1, 0))
