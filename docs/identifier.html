Identifier (component object)

entity:add(Identifier c, String name = "", String layer = "")

Provides a way to name individual entities, and groups of entities, with understandable string identifiers.

The identifier component can be added with two constructor argument strings, a name that should be unique, and a layer that can be shared.
An empty string will result in that name or layer not being assigned.

local entity = Entity.create()

entity:add(Identifier, "unique_name", "shared_layer")

Entity Identifier.getByName(String name)

Returns an Entity object that has the unique given name.

local entity = Identifier.getByLayer("unique_name")

Entity Identifier.getByLayer(String layer, Int number = 0)

Returns one of many Entity objects that have the given layer identifier.
The number argument is used to select which entity from the layer to return.

local entity = Entity.create()
entity:add(Identifier, "", "shared_layer")

local entity = Identifier.getByLayer("shared_layer")

Bool Identifier.hasName(String name)

Returns if an entity with the given unique name exists.

local entity = Entity.create()
entity:add(Identifier, "unique_name")

local has = Identifier.hasName("unique_name")

Int Identifier.hasLayer(String layer)

Returns how many entities have the given layer identifier.

local number = Identifier.hasLayer("shared_layer")

identifier:rename(String name, String layer)

Allows an identifier component to have its name and layer reset.

local entity = Entity.create()
entity:add(Identifier, "", "shared_layer")
local identifier = entity:get(Identifier)

identifier:rename("new_name", "new_layer")

String identifier.name

Returns the name string of the identifier component, an empty string if unassigned.

local entity = Entity.create()
entity:add(Identifier, "", "shared_layer")
local identifier = entity:get(Identifier)

print(identifier.name)

String identifier.layer

Returns the layer string of the identifier component, an empty string if unassigned.

local entity = Entity.create()
entity:add(Identifier, "", "shared_layer")
local identifier = entity:get(Identifier)

print(identifier.layer)
