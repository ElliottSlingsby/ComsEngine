Mesh (component object)

entity:add(Mesh c, String file)

Provides a way to attach a wavefront obj mesh file to an entity. Entity requires a transform component.

The mesh component can be added with the constructor argument with the file location of an obj mesh file.

If the mesh has already been loaded it have to load it again, otherwise it will load the mesh during run-time.

local entity = Entity.create()

entity:add(Mesh, "mesh/cube.obj")

mesh:reload(String name)

Allows a mesh to be changed to another obj mesh file.

If the mesh has already been loaded it have to load it again, otherwise it will load the mesh during run-time.

local entity = Entity.create()
entity:add(Mesh, "mesh/cube.obj")

entity:get(Mesh):reload("mesh/sphere.obj")
