Input (core object)

Input.addInput(String name, Int scancode)

Binds a keyboard scancode to a string.

A full list of scancodes can be found in SDL_scancode.h.

Input.addInput("forward", 26)
Input.addInput("back", 22)
Input.addInput("left", 4)
Input.addInput("right", 7)
Input.addInput("escape", 41)

Bool Input.wasDown(String name)

Checks if a bound string was pressed in the past.
The input string will stay true until Input.wasDown has been checked, similar to an event.

local down = Input.wasDown("forward")

Bool Input.isDown(String name)

Checks if a bound string is currently being pressed.

local down = Input.isDown("forward")

Bool Input.mouseLWasDown() | Bool Input.mouseRWasDown()

Similar to Input.wasDown, but for the mouse.

local lDown = Input.mouseLWasDown() local rDown = Input.mouseRWasDown()

Bool Input.mouseLIsDown() | Bool Input.mouseRIsDown()

Similar to Input.isDown, but for the mouse.

local lDown = Input.mouseLIsDown() local rDown = Input.mouseRIsDown()

Vec3 Input.mousePos()

Returns the mouse position relative to the top left corner of the screen.

local vector = Input.mousePos()

Vec3 Input.mouseRelativePos()

Returns the difference in mouse position relative to the centre from the previous frame.

local vector = Input.mouseRelativePos()

Input.lockMouse(Bool locked)

Locks and hides the cursor to the window.

Input.lockMouse(true)
