The Console

The CMD window of the engine can be used as a console to execute single lines of Lua, providing no scripts are spamming text to it.

Use it by assigning global variables.

Console with no input.

>>>


Creating a global variable.

>>> entity = Entity.create()
>>>


Using global variable

>>> entity = Entity.create()
>>> entity:add(Transform)
>>>


Using print

>>> entity = Entity.create()
>>> entity:add(Transform)
>>> print(entity)
userdata: 000000B5E252DE78
>>>
