Engine (core object)

Number Engine.dt()

Provides previous tick-time in floating point seconds.
Use as a multiplier to make data transformations relative to tick rate.

local number = 100
number = number * Engine.dt()

Engine.shutdown()

Shut down the engine after the current tick is finished.

Engine.shutdown()

Entity Engine.camera() | Engine.camera(Entity camera)

Get and set the camera entity that the renderer will use as the main point-of-view.

Camera starts as default set as an entity with a transform component.
Any new camera must also have a transform component.

local oldCamera = Engine.camera()

local newCamera = Entity.create()
newCamera:add(Transform)

Engine.camera(newCamera)

String Engine.config(String key) | Engine.config(String key, String value)

Get and set engine configuration strings to re-define the input of some systems.
Engine.config only works from within Load.lua.

local oldTitle = Engine.config("window.title")

Engine.config("window.title", "new title")


Below shows available configuration keys on the left, and their default values on the right.

"script.start" = "Start.lua"
"window.title" = "ComsEngine"
"window.x" = "720"
"window.y" = "405"
"shader.vertex" = "Vertex.gls"
"shader.fragment" = "Fragment.gls"
